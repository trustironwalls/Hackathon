Welcome to KaliLab 🐉

An interactive browser-based lab for learning Linux navigation inside Kali Linux — from your very first ls to scripting and privilege enumeration.

Pick a lesson from the sidebar to begin. Each lesson has:

  • 📖 A short lesson with real commands
  • 💻 A live terminal simulator — type commands and see output
  • 🎯 A challenge task to complete before moving on

The terminal simulates a real Kali Linux filesystem. Try ls, pwd, or whoami right now!
